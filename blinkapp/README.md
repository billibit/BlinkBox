# BlinkBox Product App

`blinkapp/` will contain the Next.js customer/admin product application.

This workspace owns:

- Customer onboarding, budget setup, preferences, gift history, feedback, and support.
- Admin catalog, decision review, order, fulfilment, refund, and support workflows.
- Route Handlers, Server Actions, domain services, validation, Drizzle schema, migrations, app scripts, and tests.
- Stripe SetupIntent/Checkout setup mode onboarding and PaymentIntent per-gift charging after admin approval.

The app remains the authority for approvals, payments, orders, fulfilment state, support, and audit history.
